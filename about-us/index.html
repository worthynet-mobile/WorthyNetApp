My Vision

Keep everyone up to date.

Lost connection or going above the limit of the internet package are common question that people face in day-to-day life. So, the next best option is to get a hotspot from a person near you to fulfil the work, but the other way round the hotspot share may face to some cost by sharing the hotspot. So, what about if there is a way to fulfill the needs of both sharer and receiver, the solution is the WorthyNet.

My Mission

Easy, Reliable, and Safe way of sharing your network.

WorthyNet is an Android 9 based mobile application that gives a totally new experience in hotspot sharing. The main goal of this application is to give the chance to the user who share the hotspot to get a fee from its user that takes the hotspot. The only needs to use this service are the both sharer and receiver have to install the application in the mobile and need to have a PayHere account. In the application there are in-built time-based packages that covers commonly needs. So, the user can assign a price for each package at his own will. So the receiver can select a package from that and can activate that package so the both needs of sharer and receiver are fulfilled.
